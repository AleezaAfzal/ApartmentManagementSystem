This project is an Apartment Management System built with .NET 8. It helps tenants and owners manage apartments efficiently. The system is still under development, adding extra functionalities and improving my programming and software development skills.

## Description
This system allows tenants and owners to interact with apartment-related services.  

Guests can browse and search apartments for rent on the website.  
To book a visit, guests must sign up and create an account.  
Once signed up and approved, users become tenants and can log into the tenant portal to access additional features.  
Owners have their own portal to manage buildings, apartments, services, bills, and tenant complaints.  

## Current Features

### Guest Features
Browse and search apartments without signing in  
Booking visits available after signing up  

### Tenant Features
Tenant login  
Book facilities like garden hall  
Upload payment receipts  
Submit complaints  
Add reviews  

### Owner Features
Owner login  
Add building (currently only one)  
Add apartments for rent  
Issue bills  

## Planned Features
Allow tenants to access more services through the portal  
Owners can add all services and generate automatic bills  
Owners can update tenants on ongoing complaint status  
Improve usability and functionality for both tenants and owners
